Getting Started

The FusedVR ChainAuth APIs are designed to be as simple as possible to integrate into any game or application.

First you will need an Application ID for your project. As this project is in beta, you will need to request an application id. As the next phase of this project, we intend to have a dashboard where you can create, manage, and gain insights into your application. For the time being, please request an application id on the FusedVR Discord in the crypto channel.

Once you have an Application ID, you can start using the APIs or the Unity SDK to start making requests and authenticating players.

For a guide on how to use the Unity SDK, please refer to the Hello World tutorial.
